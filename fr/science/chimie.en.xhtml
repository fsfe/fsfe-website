Free softwares for chemistery (Listed by Jérôme Pansanel.)

Name Description URL
AMMP a modern full-featured molecular mechanics, dynamics and modeling program. http://www.cs.gsu.edu/~cscrwh/ammp/ammp.html
ChemCalc a calculator that is more oriented towards chemistry. http://chemcalc.sourceforge.net/
Chemeq translate chemical equations to LaTeX code. ftp://boltz.univ-littoral.fr/pub/chemeq/
Chemsuite molecular 2/3D drawing and chemical reaction simulation software. http://chemsuite.seul.org/
CHIMP a generic tool for the modeling of chemical phenomena. http://chimp.sourceforge.net/
CRK The CRK (Chemical Resource Kit) is the collective name describing three chemistry-related software packages. http://crk.sourceforge.net/
DND intend to build a GUI for molecular dynamics simulation of molecule. http://theopenlab.uml.edu/dnd/
Garlic a free/open source molecular visualization program for UNIX/Linux. http://pref.etfos.hr/garlic/
gdpc a program for visualising molecular dynamic simulations. http://www.physics.helsinki.fi/~frantz/software/gdpc.html
gdis a GTK based program for visualizing molecules and periodic systems. http://gdis.seul.org/
Ghemical a molecular modelling software package with nice 3D-visualization tools. http://www.uku.fi/~thassine/ghemical/
GPeriodic a periodic table reference software. http://gperiodic.seul.org/
GtkChemkin handles save-files written by the chemkin program. http://kruemel.mine.nu/~nilsb/
JChemPaint a program for drawing 2D chemical structures, written in Java. http://jchempaint.sourceforge.net/
JCML Java library for utilizing Chemical Markup Language (CML). http://www.openscience.org/~egonw/cml/
Jmol an open-source molecule viewer and editor. http://jmol.sourceforge.net/
kc transforms chemical equations into simulation programs. ftp://fatou.ruc.dk/pub/kneth/
Kemistry a collection of chemistry-related applications for KDE. http://kemistry.sourceforge.net/
KMol KMol is a molecular weight and elemental composition calculator. http://www.idiom.com/~tomi/kmol.html
KMovisto KMovisto is a molecule viewer for using in quantum chemistry. The program reads GAUSSIAN 94 and GAUSSIAN 98 files (obtained from UNIX or MS Windows systems) or XYZ files and draws the molecule as a ball and stick model. http://members.tripod.de/PageOfMH
Lennard Nobel Gas Simulator http://metalab.unc.edu/pub/Linux/science/chemistry/
Moiss a n-particle, n-dimensions Monte Carlo Schroedinger equation integrator http://sourceforge.net/projects/moiss/
Moldy Moldy is a general-purpose molecular dynamics simulation program. It is sufficiently flexible that it ought to be useful for a wide range of simulation calculations of atomic, ionic and molecular systems. http://www.earth.ox.ac.uk/~keith/moldy.html
MOSCITO MOSCITO is designed for molecular dynamics simulations of condensed phase systems defined by classical molecular mechanics force-fields such as AMBER, OPLS, CHARMM and GROMOS. http://ganter.chemie.uni-dortmund.de/~pas/moscito.html
MPQC MPQC is the Massively Parallel Quantum Chemistry Program. http://aros.ca.sandia.gov/~cljanss/mpqc/
NanoCAD NanoCAD uses mathematical techniques of molecular modeling to simulate the behavior of molecules on your web browser. http://world.std.com/~wware/ncad.html
ORAC ORAC is a program for running classical simulations of biomolecules. Simulations can be carried out in the NVE, NPT, NHP, and NVT thermodynamic ensembles. http://www.chim.unifi.it:8080/orac/
OSET Computer-Assisted Organic Synthesis (CAOS) program for use in the teaching of Organic Chemistry, and retrosynthetic analysis in particular. http://litio.pquim.unam.mx/caos/
VASP Viewer This is a scientific visualization package for examining output files generated by the Vienna Ab-initio Simulation Package, a package for performing ab-initio quantum-mechanical molecular dynamics using pseudopotentials and a plane wave basis set. http://vaspview.sourceforge.net/
XMakeMol a program for viewing and manipulating atomic/molecular systems given in xyz format. http://vegemite.chem.nottingham.ac.uk/~xmakemol/
xyz2rgb xyz2rgb makes pretty pictures and movies of molecules. It expects input in the form of an XYZ file, one of the file formats frequently used by computational chemists. As output, it produces one or more line-interlaced RGB files. http://world.std.com/~wware/xyz2rgb.html
Chemtools Chemtool is a small program for drawing chemical structures on Linux and Unix systems using the GTK toolkit under X11. http://ruby.chemie.uni-freiburg.de/~martin/chemtool/chemtool.html
mwcalc This program is a Molecular Weight Calculator (console tools). http://pansanel.multimania.com/linux/mwcalc.html
MMTK The Molecular Modelling Toolkit (MMTK) is an Open Source program library for molecular simulation applications. In addition to http://dirac.cnrs-orleans.fr/MMTK/
