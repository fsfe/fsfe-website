Die Aufgabe der Freedom Task Force ist es, die Freie-Software-Gemeinschaft zu unterstützen, und Sie können uns dabei helfen. Auch wenn die FSFE einen Vollzeit-Koordinator für die FTF bereitstellt, wird ein Großteil unserer Arbeit mit der Hilfe von Freiwilligen bewältigt.

Sie wollen helfen? Dann melden Sie sich auf unserer Mailingliste an. Sie können uns zum Beispiel helfen, indem Sie Informationsmaterial zu Lizenzfragen rund um Freie Software erstellen oder indem Sie Informationen über die FTF verbreiten.
