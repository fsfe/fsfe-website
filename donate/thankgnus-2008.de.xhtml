Liste der Unterstützer in 2008

Die Free Software Foundation Europe möchte allen Freiwilligen, Helfern und anderen danken, die uns bei unserer Arbeit für Freie Software unterstützt haben. Informationen, wie Sie uns in unserer Arbeit für Freie Software unterstützen können, finden Sie hier. Details für potentielle Spender sind ebenfalls verfügbar.

Die Free Software Foundation Europe möchte namentlich den folgenden Spendern danken. Die Unterstützung von Ihnen macht es uns möglich, weiterhin für Freie Software aktiv zu sein.

Wir möchten zudem allen Spendern danken, die es vorgezogen haben, anonym zu bleiben. Wenn auch Sie spenden möchten, finden Sie alle notwendigen Informationen hier. Sollten Sie auf der Suche nach den Sponsoren-Grafiken sein, so finden Sie diese hier.

Hardware Spenden werden auf unserer Hardware Spenden Internetseite aufgelistet. Dort finden sie außerdem Informationen, wie Sie uns mit Hardware unterstützen können.
