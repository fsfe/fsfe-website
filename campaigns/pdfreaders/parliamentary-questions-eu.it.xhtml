Software Libero per leggere PDF

Interrogazioni parlamentari

2 dicembre 2011 - E-011374/2011 - Interrogazione con richiesta di risposta scritta alla Commissione Articolo 117 del regolamento
Jan Philipp Albrecht (Verts/ALE) , Françoise Castex (S&D) , Marietje Schaake (ALDE) , Christian Engström (Verts/ALE) and Francisco Sosa Wagner (NI)
Copyright: European Union, 2011 - Source: European Parliament

Numerosi siti web pubblici dell'UE rimandano a software proprietari, quali per esempio Acrobat Reader, come unica possibilità di visualizzazione dei file PDF e forniscono un link per scaricare il programma. Si tratta di casi di pubblicità diretta da parte delle istituzioni pubbliche che crea un vantaggio competitivo a favore di un'impresa per la vendita dei suoi prodotti.

  1. Su quante pagine web della Commissione appaiono attualmente tali pubblicità?
  2. Potrebbe la Commissione fornire i link di tali pagine, nonché i nomi delle istituzioni europee sulle cui pagine web appare tale tipo di pubblicità?
  3. Per quali motivi pubblicizza la Commissione questo specifico software come unico possibile mezzo di visualizzazione dei file PDF (per esempio profitto di natura finanziaria, mancanza di altro software funzionale, ecc.)?
  4. Hanno provveduto le istituzioni sui cui siti web appare tale pubblicità a chiarire se hanno precedentemente provato altri programmi di lettura di file PDF? Quali software hanno indicato?
  5. Quali azioni sta adottando la Commissione al fine di risolvere tale problema che riguarda a) lo stesso sito della Commissione, e b) i siti web delle istituzioni pubbliche degli Stati membri?
