Εξαιτίας των σημαντικών τραπεζικών προμηθειών για διεθνείς χρηματικές μεταφορές, συχνά δεν έχει νόημα μικρές δωρεές ή διαταγές πληρωμών να αποστέλλονται άμεσα στον τραπεζικό λογαριασμό του FSFE.

Συνεπώς μπορείτε επίσης να συνεισφέρετε μέσω πιστωτικής κάρτας ή με άλλο τρόπο πληρωμής αποδεκτό από την PayPal. Η PayPal δεν έχει έξοδα έναρξης, μηνιαίες χρεώσεις ή δασμολογικά έξοδα, ωστόσο χρεώνει 1.9% + $0.30 USD για κάθε συναλλαγή.

Οπότε θα θέλαμε να σας ζητήσουμε να στείλετε μήνυμα στη διεύθυνση <office@fsfeurope.org> και να μας γνωστοποιήσετε πού να στείλουμε ευχαριστήρια επιστολή. Επίσης θα θέλαμε να σας καταχωρίσουμε στη σελίδα μας ''Thank GNUs''. Ακόμη και αν προτιμάτε να παραμείνετε ανώνυμοι, παρακαλούμε γνωστοποιήστε μας πώς μπορούμε να απευθυνθούμε σε σας, για να σας ευχαριστήσουμε κατ' ιδίαν.

Αν ενδιαφέρεστε για δημόσια ανακοίνωση της συνεισφοράς σας, το FSFE επίσης παρέχει μερικά κουμπιά χορηγιών για τις δωρεές προς αυτό.
