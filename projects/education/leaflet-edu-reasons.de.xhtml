Gründe für die Schulen, Freie Software zu bevorzugen

Zugang zu Software entscheidet, wer an der digitalen Gesellschaft teilnehmen darf. Die Freiheiten, Software zu nutzen, zu studieren, zu verteilen und zu verbessern ermöglichen eine gleichberechtigte Teilhabe und sind daher extrem wichtig. Es gibt viele Gründe dafür, Freie Software im allgemein zu nutzen. Hier sammelen wir einige, die besonders wichtig für Schulen sind:

Teilen

Der Einsatz Freier Software ermöglicht Schulen, Kindern beizubringen, zu teilen und zu kooperieren. Sie treten damit eine ganze Gemeinde bei, bei der das Wissen verteilt wird.

Gleichheit zuhause

Bei Freier Software können Lehrer eine Kopie an jeden Schüler geben. So können Kinder aus Familien mit weniger finanziellen Mitteln mit den gleichen Werkzeugen wie jedes andere Kind lernen.

Programmieren lernen

Die Möglichkeit zu basteln motiviert Kinder dazu, mehr zu erfahren. Freie Software ermöglicht es, über Computer in beliebiger Tiefe zu erfahren.

Learning to use software

Free Software permits children to learn how software works and thus to understand the concepts underlying a whole category or type of software. They are then able to adapt to any environment, a key skill nowadays.

Frie Software in der Muttersprache

Free Software-Anwendungen werden in fast alle Sprachen übersetzt. So kann jedes Kind lernen und sich auf das eigentliche Thema konzentrieren one Sprachen Probleme oder kulturelle Barrieren zu erleben.

Unabhängigkeit

Wenn Kindern unterrichtet wird, wie man mit proprietären Software umgeht, werden sie an etwas abhängig gemacht für was sie später werden bezahlen müssen. Wenn die Kinder lernen, mit Freien Software umzugehen, können sie weiterhin dieser Software in ihrem Leben als Erwachsener benutzen, und sie können auch dabei anderen helfen, indem sie das Software verteilen dürfen.

Kürzen Sie die Hardware-Kosten drastisch

Freie Software kann mit minimalen Hardware-Anforderungen weiter verwendet werden, und zwar auf fast jeder Hardware. Alte Computer erhalten dadurch ein neues Leben, was gleichzeitig Geld spart und die Umwelt schützt.

Vergessen Sie Ihre Lizenzen Kosten

Ohne Lizenzgebühren, spart die Schule Geld und kann es verwenden, um Lehrer zu trainieren oder um zB den technischen Support zu bezahlen, und die Lehrer dadurch zu entlasten.

Machen Sie sich keine Sorgen wegen Lizenzen

Bei Freier Software gibt es keine Sorgen über Verfallsdaten oder Kosten der Lizenzen. Freie Software bleibt in jeder Hinsicht frei. Allen Kindern und Studenten stehen legale Kopien zu Verfügung

Einfach zu verwalten

Freie Software ist stabil, sicher und zuverlässig. Sicherheit und andere Updates können in einer Minute durchgeführt werden. Seine Wartung ist automatisiert und somit zeitsparend.
