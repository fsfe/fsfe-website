¿Qué es la Freedom Task Force?

Es un protector del Software Libre. La Freedom Task Force ofrece servicios de soporte de licencias a personas particulares, proyectos y negocios envueltos con Software Libre. Estamos aquí para ayudar a consolidar la fundación legal del ecosistema de Software Libre

¿Qué servicios ofrece la Freedom Task Force?

  • Actividades fiduciarias
  • Violación de licencias
  • Formación en materia de licencias

¿Cómo puedo contactar a la Freedom Task Force?

  • Ud. puede enviar un mensaje a través denuetro formulario en línea
  • Nos puede enviar un email a ftf@roundup.fsfeurope.org

¿Qué conección tiene con gpl-violations.org?

Trabajamos en sociedad con gpl-violations.org para protejer el Software Libre.

¿Qué conección tiene con la FSF License Lab?

>Todas las fundaciones de Software Libre cooperan para asegurar la promoción y protección de Software Libre.

¿Qué conección tiene ud. con el Software Freedom Law Center?

Perseguimos el mismo objetivo de promover y proteger el Software Libre. Cooperaremos en la medida de lo posible para asegurar la promoción y la protección de Software Libre.
