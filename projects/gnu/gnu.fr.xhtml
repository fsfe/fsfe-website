GNU n'est pas Unix !


Le projet GNU a été annoncé en 1983 par Richard M. Stallman et est sorti en 1984 pour créer un système d'exploitation clone d'Unix à base de Logiciel Libre. Des variantes de ce système, utilisant le noyau Linux, sont utilisées à grande échelle autour de la planète. Ces systèmes GNU/Linux sont la base de toutes les distributions dites "Linux" de nos jours.

Dans l'esprit du projet GNU, la Free Software Foundation a été fondée en 1985 ; à l'origine pour s'occuper des besoins légaux du projet GNU.

La FSFE soutient la FSF Nord Américaine pour son travail soutenu sur le projet GNU, autant dans les domaines techniques que juridiques.

La page d'accueil du projet GNU est 

   http://www.gnu.org
La page d'accueil de la Free Software Foundation à Boston est 
   http://www.fsf.org
