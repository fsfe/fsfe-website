Software Livre para ler PDF

Perguntas Parlamentares

2 de Dezembro de 2011 - E-011374/2011 - QPergunta com pedido de resposta escrita à Comissão Artigo 117.º do Regimento
Jan Philipp Albrecht (Verts/ALE) , Françoise Castex (S&D) , Marietje Schaake (ALDE) , Christian Engström (Verts/ALE) and Francisco Sosa Wagner (NI)
Copyright: European Union, 2011 - Source: European Parliament

Inúmeros sítios Web da UE remetem para software proprietário, como o Acrobat Reader, como o único meio possível para leitura de ficheiros PDF e fornecem uma hiperligação para o seu descarregamento. Este é um caso de publicidade directa feita por instituições públicas, que cria uma vantagem competitiva para uma empresa vender os seus produtos.

  1. Actualmente, em quantas páginas Web da própria Comissão surgem estes anúncios?
  2. A Comissão pode fornecer as hiperligações para essas páginas, bem como os nomes de instituições da UE que expõem essa publicidade nos seus sítios Web?
  3. Quais são as razões da Comissão (por exemplo, ganho financeiro, inexistência de outro software funcional, etc.) para anunciar este software específico como o único meio possível para a leitura de ficheiros PDF?
  4. As instituições que expõem essa publicidade no seu sítio Web indicaram se anteriormente experimentaram outros leitores de PDF? Que software indicaram?
  5. WQue medidas tenciona tomar a Comissão para resolver este problema em relação a) ao sítio Web da própria Comissão, e b) aos sítios Web de instituições públicas dos Estados-Membros?
