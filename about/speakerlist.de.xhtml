Redner der FSFE

"Frei wie Redefreiheit"

Wenn Sie einen Redner suchen, der die Free Software Foundation Europe repräsentieren kann und die Freie-Software-Bewegung in einer Konferenz, einer Veranstaltung oder einer Podiumsdiskussion vertreten soll, ist diese Seite für Sie gemacht. Wenn Sie einen dieser Redner einladen möchten, dann kontaktieren Sie den Redner via E-Mail.

Bernhard Reiter

Bernhard Reiter ist außergewöhnlich gut im Erklären, wie man ein ethisches Geschäft mit Freier Software aufbauen kann. Er kann auch über Freie-Software-Entwicklung, die Finanzierung Freier Software und über die Geschichte der Free Software Foundation Europe reden.

Sprachen

Er spricht Englisch und Deutsch.

Georg C.F Greve

Georg C.F Greve kann über Freie Software aus verschiedenen Blickwinkeln diskutieren: Über die philosophischen, sozialen, wissenschaftlichen, ökonomischen, politischen und zur Bildung gehörenden Aspekte. Er kann auch Reden über Offene Standards, politische Herausforderungen der digitalen Gesellschaft und über die Free Software Foundation Europe halten.

Sprachen

Georg spricht fließend Englisch und Deutsch.
