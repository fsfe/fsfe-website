Ελευθερία

Τι κοινό έχουν το Ελεύθερο Λογισμικό με τα εργαλεία

Μια σύγκριση

Το Ελεύθερο Λογισμικό μάς προσφέρει βασικές ελευθερίες που άλλοι τύποι λογισμικού μας αρνούνται. Οι πιο πολλοί από εμάς συνηθίσαμε σε σοβαρούς ψηφιακούς περιορισμούς, και γι' αυτό μία αναλογία με ένα πολύ συνηθισμένο φυσικό εργαλείο είναι χρήσιμη.
Το Ελεύθερο Λογισμικό μπορεί να χρησιμοποιηθεί σαν εργαλείο. Ας προσπαθήσουμε να ζωγραφίσουμε μια εικόνα με τις ανήκουστες επιτώσεις που θα σημειωνόταν εάν τα κατσαβίδια μπορούσαν να χρησιμοποιηθούν μόνο με τους ίδιους περιορισμούς που ισχύουν και για το μη-ελεύθερο λογισμικό:

  • Τα κατσαβίδια δεν θα ήταν πια προτυποποιημένα και θα λειτουργούσαν μόνο με τις βίδες που θα έφτιαχνε η ίδια εταιρεία.
  • Οι άδειες για τα κατσαβίδια θα επέτρεπαν μόνο το ξεβίδωμα της βίδας! Για να σφίξουμε μια βίδα, θα χρειαζόμασταν ένα πιο ακριβό κατασαβίδι, πάντα από τον ίδιο κατασκευαστή.
  • Θα επιτρεπόταν μόνο σε μια εταιρεία να κατασκευάζει τα κατσαβίδια, επειδή η ιδέα θα είχε πατενταρισθεί με διπλώματα ευρεσιτεχνίας. Μόνο μια πολυδάπανη δοκιμή θα μπορούσε να οδηγήσει σε γενίκευση του δικαιώματος χρήσης, για παράδειγμα, μιας τανάλιας για το σφίξιμο ή για το ξέσφιγμα μιας βίδας.
  • Δεν θα μας επιτρεπόταν να τροποποιήσουμε τα κατσαβίδια που αγοράσαμε για να τα προσαρμόσουμε στις ανάγκες μας. Θα απαγορευόταν, για παράδειγμα, να τυλίξουμε υφασμάτινη ταινία γύρω από την λαβή για να μη γλυστρά το κατσαβίδι από τα χέρια μας.
  • Θα γινόταν παράνομο να επισκευάσουμε ένα χαλασμένο κατσαβίδι χωρίς την άδεια του κατασκευαστή. Δεν θα μπορούσαμε ούτε καν να το αποσυναρμολογήσυμε για να μελετήσουμε πως δουλεύει.
  • Θα έβγαιναν συνεχώς νέα κατσαβίδια και νέες βίδες στην αγορά. Δεν θα ήταν όμως συμβατά με τα παλιότερα μοντέλα. Οπότε, θα ήμασταν συνεχώς αναγκασμένοι να αγοράζουμε καινούργια κατσαβίδια για να συντηρούμε τις νέες μας συσκευές και τα μηχανήματα.
  • Θα απαγορευόταν επίσης να δανείζουμε ή να μεταπωλούμε τα κατσαβίδια.
  • Τα κατσαβίδια θα μας κατασκόπευαν στα κρυφά, και θα έστελναν, στα ύπουλα, τις πληροφορίες που συλλέγουν προς εξωτερικά πρόσωπα (ή εταιρείες, ή κυβερνήσεις), χωρίς εμείς να το ξέρουμε.

Κανένας δεν θα χρησιμοποιούσε κατσαβίδια αν η χρήση τους τελούσε υπό τέτοιους περιορισμούς. Δεν θα υπήρχε εύλογο έδαφος για να αποδεχθεί κανείς τέτοιους όρους. Όχι απλώς για τα εργαλεία, αλλά ούτε και για το λογισμικό.

Οι ελευθερίες που θεωρούμε αυτονόητες με τα εργαλεία, είναι επίσης εφικτές και με τους υπολογιστές. Τα ελεύθερα προγράμματα μπορούν να εγκατασταθούν οπουδήποτε χωρίς καμία ανησυχία. Αφού τα Ελεύθερο Λογισμικό βασίζεται στα Ανοικτά Πρότυπα, θα μπορείτε να ανοίγετε τα αρχεία σας με διάφορα προγράμματα, ακόμη και μετά από πολά χρόνια.

Από την άλλη μεριά, τα διπλώματα ευρεσιτεχνίας στο λογισμικό (πατέντες), η Διαχείριση Ψηφιακών Περιορισμών (DRM), τα κλειστά πρότυπα, οι ιδιοταγείς άδειες χρήσης λογισμικού, καθώς και οι άδικοι όροι χρήσης, είναι όλα τρόποι περιορισμού της ελευθερίας των χρηστών.

Που χρησιμοποιείται το Ελεύθερο Λογισμικό;

Σε επιτραπέζιους και φορητούς υπολογιστές: Υπάρχει μια απίστευτη ποικιλία Ελεύθερου Λογισμικού που είναι διαθέσιμο για όλες τις δουλειές σας, τα σχεδιά σας και την καθημερινή σας χρήση. Ακολουθούν μερικά παραδείγματα:

Το GNU/Linux είναι ένα ελεύθερο Λειτουργικό Σύστημα. Διατίθεται σε πολλές διαφορετικές ποικιλίες, που λέγονται διανομές. Μερικές είναι πολύ πιο κατάλληλες για αρχάριους, ενώ άλλες δημιουργήθηκαν για να καλύψουν εξειδικευμένες επαγγελματικές ανάγκες.

Ο Mozilla Firefox, το VLC, το Audacity, και το LibreOffice είναι παραδείγματα ευρέως διαδεδομένων προγραμμάτων Ελεύθερου Λογισμικού, αλλά υπάρχουνν και χιλιάδες άλλα.

Στις άλλες σας συσκευές: Έχε γίνει συνήθεια να κουβαλάμε μαζί μας διάφορους υπολογιστές, όλη την ώρα. Τους αποκαλούμε 'έξυπνα τηλέφωνα', 'ταμπλέτες', ή 'αναγνώστες ηλεκτρονικών βιβλίων (e-readers)'. Στην πραγματικότητα, δεν είναι άλλο παρά υπολογιστές, οι οποίοι, συχνά, έχουν υπερβολικούς περιορισμούς.

Τα κινητά τηλέφωνα Android είναι ένα παράδειγμα συσκευών που μπορούν να ελευθερωθούν. Για παραπάνω πληροφορίες, δείτε το https://freeyourandroid.org

Σε όλους τος υπολογιστές, παντού τριγύρω μας:Ο κατάλογος των βασικών ηλεκτρονικών συσκευών που ελέγχονται από υπολογιστές, οι οποίοι, με τη σειρά τους, ελέγχονται από λογισμικό, είναι σχεδόν ατελείωτη. Σε μερικές από αυτές τις συσκευές, χρησιμοποιείται ακόμη ιδιοταγές λογισμικό. Ευτυχώς, πάντως, το Ελεύθερο Λογσιμικό αντιπροσωπεύει ένα μεγάλο ποσοστό των ενσωματωμένων εφαρμογών.

φούρνοι, τηλεοράσεις, συστήματα θέρμανσης, συσκευές αναπαραγωγής CD, αυτοκίνητα, τραίνα, πλυντήρια πιάτων, προβολείς, τηλε-κατευθυνόμενα ελικόπτερα ...

Η υποδομή του Internet βασίζεται, επίσης, ευρέως στο Ελεύθερο Λογισμικό, όπως εξάλου και το 'web'.

modem, διακομιστές DNS, διακομιστές αλληλογραφίας, δρομολογητές (routers), βάσεις δεδομένων, λογισμικό για ηλεκτρονικό εμπόριο, ιδιωτικά εικονικά δίκτυα VPN, διαδικτυακοί διακομιστές

Ελεύθερα με την έννοια της λέξης ελευθερία, και όχι ελεύθερα με την έννοια της δωρεάν μπύρας

Δεν χρειάζεται να πληρώσετε για τα περισσότερα προγράμματα Ελεύθερου Λογισμικού. Αρκετά προγράμματα ιδιοταγούς λογισμικού είναι επίσης χωρίς χρέωση, αλλά το Ελεύθερο Λογισμικό δεν έχει σχέση με την τιμή: αφορά την Ελευθερία.

Όταν δεν ελέγχετε ένα πρόγραμμα, τότε το πρόγραμμα ελέγχει εσάς. Συνεπώς, όποιος ελέγχει το λογισμικό, ελέγχει και εσάς.

Για παράδειγμα, κανείς δεν επτρέπεται να μελετήσει πως λειτουργεί ένα μη-ελεύθερο πρόγραμμα και τι ακριβώς κάνει στον υπολογιστή σας ή στο τηλέφωνό σας. Μερικές φορές, οι ατέλειες του μη-ελεύθερου λογισμικού συνίστανται απλώς στο ότι δεν μπορεί να ανταποκριθεί στις δικές σας προσδοκίες. Το μη-ελεύθερο λογισμικό, πάντως, τις περισσότερες φορές, επιτρέπει τη διαρροή των δεδομένων σας ή υποκρύπτει κακόβουλα χαρακτηριστικά.

Αν τρέχετε αποκλειστικά Ελεύθερο Λογισμικό στον υπολογιστή σας ή στη συσκευή σας, έχετε τον πλήρη έλεγχο. Μπορεί ακόμη και να μην έχετε τις δεξιότητες για να ασκήσετε και τις τέσσερες ελευθερίες από μόνοι σας, αλλά θα επωφεληθείτε από τη γνώση και την ειδημοσύνη των εξαιρετικά ζωντανών κοινοτήτων.

Σκεφθείτε να επιδείξετε την εκτίμησή σας αμοίβοντας εκουσίως τους προγραμματιστές. Με αυτό τον τρόπο εξασφαλίζετε, επίσης, πως εσείς είστε ο πελάτης και όχι το προϊόν.

Τι είναι το Ελεύθερο Λογισμικό;

  1. Χρήση: Λογισμικό που μπορεί ο καθένας να το εκτελεί ελεύθερα, για οποιονδήποτε σκοπό.
  2. Μελέτη: Λογισμικό που μπορεί ο καθένας να επιθεωρεί και να το προσαρμόζει στις ατομικές του ανάγκες ή επιθυμίες.
  3. Διαμοιρασμός: Λογισμικό που μπορεί ο καθένας να αντιγράφει και να το μοιράζεται ελεύθερα.
  4. Βελτίωση: Οι βελτιωμένες εκδόσεις του λογισμικού μπορούν να αναδιανέμονται: όλη η κοινότητα θα επωφεληθεί από τις βελτιώσεις.

Για να μπορεί ένα πρόγραμμα να είναι στα αλήθεια Ελεύθερο Λογισμικό, πρέπει να παρέχει και τις τέσσερες αυτές ελευθερίες, σε όλους.

Σχετικά με το Ευρωπαϊκό Ίδρυμα Ελεύθερου Λογισμικού (FSFE)

Αυτό το φυλλάδιο εκτυπώθηκε από το Ευρωπαϊκό Ίδρυμα Ελεύθερου Λογισμικού (FSFE), μια μη κερδοσκοπική οργάνωση, αφιερωμένη στην προώθηση του Ελεύθερου Λογισμικού, η οποία εργάζεται για να χτίσει μια ελεύθερη ψηφιακή κοινωνία.

Η πρόσβαση στο λογισμικό καθορίζει το πως εμείς μπορούμε να συμμετέχουμε στην κοινωνία μας. Έτι, το FSFE είναι αφοσιωμένο να εξασφαλίζει την ισότιμη πρόσβαση και συμμετοχή στην εποχή της πληροφορίας, μαχόμενο υπέρ της ψηφιακής ελευθερίας.

Κανείς ποτέ δεν πρέπει να υποχρεώνεται να χρησιμοποιεί λογισμικό που δεν μπορεί να χρησιμοποιηθεί, να μελετηθεί, να διαμοιράζεται και να βελτιώνεται. Πρέπει να έχουμε το δικαίωμα να προσαρμόζουμε την τεχνολογία για να καλύπτει τις ανάγκες μας.

Το έργο του FSFE επιτελείται χάρη στις προσπάθειες μιας κοινότητας ανθρώπων που είναι αφιερωμένοι σε αυτούς τους σκοπούς. Αν θέλετε να έρθετε μαζί μας και να μας βοηθήσετε να τους πετύχουμε, υπάρχουν πολλοί τρόποι για να συνεισφέρετε, ανεξάρτητα από το υπόβαθρο του καθενός: https://fsfe.org/contribute
