SFP#13: Freies Software Spiel "0 A.D. Empires Ascendant" mit Stanislas Dolcini

Beim kommenden "I Love Free Software Day" werden Freie Software Spiele im Zentrum stehen. Eines der bekanntesten ist is "0 A.D.: Empires Ascendant". In dieser Episode spricht Bonnie Mehring mit Stanislas Dolcini, dem Projektleiter von 0 A.D., über das Spiel selbst und das Projekt, und wie das Spiel zur Freien Software wurde.

Diese Podcastepisode nimmt Sie mit auf eine Reise durch die Entwicklung des Spiels und die Geschichte darüber, wie es zu einem der bekanntesten Spiele der Freien Software wurde. Sie erfahren auch, wie Sie sich an 0 A.D beteiligen können. Entdecken Sie zusammen mit Stanislas und Bonnie, wie die Community hinter einem der bekanntesten Spiele der Freien Software arbeitet, wo Unterstützung gebraucht wird und welche Arten von Entwicklern und Beteiligten es gibt. Bonnie und Stanislas erzählen auch die Erfolgsgeschichte der Veröffentlichung von 0 A.D. unter einer Lizenz der Freien Software und sie sprechen über die positive Seite der Entwicklung eines Spiels der Freien Software.
