¿Por qué Software Libre?

En la era digital, donde la tecnología y el software cumplen un cada vez más importante rol en la sociedad, la política y la economía, las cuatro libertades del Software Libre son más relevantes que nunca: La libertad para usar el software sin restricciones innecesarias, para estudiar cómo funciona el software, para ser capaz de compartirlo con quienquiera que desee, y la libertad para mejorar el software para adecuarlo a nuestros deseos. Juntas, estas libertades ofrecen acceso, participación y autodeterminación para todos. En una escala mayor, para el estado de España, estas libertades proporcionan un entorno para que las administraciones públicas logren una mayour autonomía digital, alienten el crecimiento de la economía local y el potencial para transformar a España en un autor intelectual del proceso europeo de digitalización.

El Software Libre es una herramienta poderosa. Lea cómo podemos beneficiarnos del Software Libre para ...

... fortalecer nuestra sociedad

Liberando tecnología como Software Libre garantiza el derecho de todos para usar y compartir software con tu vecino, tu colega o con cualquiera. Cada persona o familia, cada institución, organización o empresa, puede beneficiarse de la misma tecnología avanzada sin exclusióny sin costes. Desde sistemas operativos a aplicaciones especializadas, la gente se beneficia del acceso abierto a la tecnología.

Cuando el estado crea y publica una aplicación, plataforma o servicio de Software Libre, también proporciona una forma de devolver al público lo que el público ha pagado indirectamente. Esta es la razón por la cual debería ser obligatorio publicar como Software Libre el software financiado con fondos públicos.

... incrementa nuestra autonomía

El mundo del Software Libre es diferente. El Software Libre es interoperable, sus datos son exportables y su código fuente es público. Eso reduce la oportunidad de formar monopolios. El usuario y finalmente el estado ganan autonomía mediante la eliminación de monopolios tecnológicos y las dependencias asociadas en favor de nuestra crítica infraestructura de comunicación y tecnología.

Dada la interoperabilidad del Software Libre, el estado is libre para cambiar suministradores y proveedores de serviciosen lugar de estar atado a un cliente en un solución cerrada de un vendedor. Un organismo público puede incluso desarrollar software por sí mismo y, una vez desarrollado, publicarlo como Software Libre para compartirlo con otras instituciones públicas. Una licencia libre garantiza que exactamente el mismo software está disponible para cualquier otra institución pública sin pagos de licencias o dependencia de ningún vendedor. De esta forma, el Software Libre ofrece al estado el poder de invertir una vez pero beneficiarse incontables veces.

El Software Libre ofrece transparencia mediante su código fuente público. La transparencia del código fuente ayuda a sus colaboradoes a mantener el código limpio. No hay forma de introducir código "secreto" dentro del software. La transparencia ayuda a la seguridad del software y evita fallos no deseados. Sólo el Software Libre puede ofrecer semejante nivel de seguridad y autonomía al mismo tiempo.

... desarrolla nuestra economía

Cada vez que el estado compra una solución no libre, con mayor probabilidad el dinero se paga a monopolios globales de software extranjeros. El Software Libre, sin embargo, le ofrece a España nuevas avenidas económicas porque el dinero puede invertirse por el contrario en la propia economía IT nacional. La interoperabilidad del Software Libre proporciona libertad al estado para cambiar de suministradores y proveedores de servicio según su deseo. No hay necesidad de un contrato especial con un suministrador concreto de un país determinado.

Como todo el mundo puede accceder al código fuente y es libre para usar y mejorar el software, el estado español es libre para contratar personas o empresas de servicios locales para adaptar el software existente a sus necesidades específicas. Inmediatamente, las administraciones públicas pueden ofrecer trabajo a los españoles para localizar, mantener, desarrollar y trabajar en su propios sistema operativo libre e interoperable, incluyendo las aplicaciones que necesiten. De esta forma, el Software Libre ofrece la posibilidad de crear empleo nacional en lugar de pagar a empresas foráneas.

El Software Libre respalda una economía descentralizada que permite la distribución de riqueza a nivel local y da empleo directamente a la gente de España. El uso y desarrollo de Software Libre ofrece a España una salida de la crisis, y una oportunidad de estar al frente de la sociedad del conocimiento y la información europea.

Eso es el porqué

Pedimos al actual y futuro gobierno de España confiar en la flexibilidad, autonomía, libertad, riqueza y transparencia que el Software Libre ofrece. Y migrar todos los sistemas estatales al Software Libre, incluyendo la administración pública y la educación.

Con un claro compromiso del gobierno español para usar y desarrollar Software Libre, el estado español puede

  • mantener y desarrollar sistemas IT estatales autónomamente al margen de empresas extranjeras, contratando en su lugar empresas y personas españolas locales que crean, mantienen y adaptan software nuevo o existente para sus própias necesidades
  • publicar y compartir el software, de forma que todas las instituciones públicas y la gente de España pueden obtener los beneficios del Software Libre
  • ejecutar software seguro y transparente, en lugar de confiar en software no revelado de compañias extranjeras
  • adaptar y descentralizar software para los diversos idiomas de España y las necesidades específicas de las comunidades autónomas españolas

Debería ser obligatorio que todo desarrollo de software, plataformas y otros recursos digitales de un organismo público realizado con dinero público fuese publicado como Software Libre.
