A propos
  

Les statuts au format TeX de la FSF France ont été signés électroniquement avec GnuPG (GPG). Pour installer GPG utiliser: 

                apt-get install gnupg
Pour plus d'information sur les signataires voir la page contact.
Procédures de vérification des signatures.
Frédéric Couchet mad@april.org. 
wget  -O frederic.txt http://www.april.org/~mad/gpg.txt
gpg --import frederic.txt
wget http://france.fsfeurope.org/about/Constitution-fr.fr.tex
wget http://france.fsfeurope.org/about/Constitution-fr.fr.tex-frederic.sig
gpg --verify Constitution-fr.fr.tex-frederic.sig Constitution-fr.fr.tex
