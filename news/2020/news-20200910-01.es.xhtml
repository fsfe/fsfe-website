Plataforma colaborativa de software libre para las administraciones - Ideación

El aumento del uso del Software Libre es un componente central para una mayor soberanía digital. Junto con una fuerte alianza de administraciones, políticos, negocios y la sociedad civil, hacemos un llamamiento por el desarrollo de un repositorio de código con Software Libre para el sector público.

El aumento del uso del Software Libre por parte de las instituciones públicas es la columna vertebral de una mayor soberanía digital. El Software Libre, también conocido como Software de Código Abierto, puede ser revisado por todos y por lo tanto se puede comprobar independientemente los aspectos de seguridad. Las aplicaciones pueden ser operadas por el usuario y adaptadas a sus necesidades. De esta manera, se puede lograr un alto grado de independencia de los proveedores individuales. Además, las administraciones tienen la oportunidad de cooperar entre sí con mayor facilidad a través de los límites organizativos.

Junto con la Open Source Business Alliance, el grupo de trabajo federado de proveedores de servicios de tecnologías de la información municipales Vitako y otros expertos, la FSFE ha desarrollado un primer concepto de repositorio de código para el sector público y ha encontrado numerosos partidarios entre las administraciones, entre políticos, en la economía y en la sociedad civil. Bajo el lema "Un lugar para el código público", el grupo de interés quiere ahora allanar el camino para un portal a través del cual la administración pública de Alemania pueda intercambiar y desarrollar conjuntamente Software Libre, también conocido como Software de Código Abierto, de manera adecuada y conforme a la ley.

Como primer resultado, se elaboró un documento de trabajo que pone en contexto la idea de un repositorio de código y describe los retos. También aclara el potencial del Software Libre para su uso en instituciones públicas y cómo dicho repositorio puede contribuir a un mayor desarrollo de la digitalización de las administraciones.

"El uso del Software Libre se impone en las instituciones públicas: El Software Libre da a todos el derecho de usar, entender, compartir y mejorar los programas para cualquier propósito. Las instituciones públicas se financian con los impuestos. Estas, deben asegurarse de gastar el dinero de la manera más eficiente posible. Al usar Software Libre, las instituciones públicas pueden colaborar en el desarrollo del código y usar las soluciones existentes sin tener que reinventar la rueda una y otra vez. ¡Si se trata de dinero público, el código también debería ser público! Además, el Software Libre ayuda a minimizar las dependencias de los proveedores individuales y así crea la base para la soberanía digital. Por lo tanto, la FSFE ha desarrollado la campaña "¿Dinero Público? ¡Código Público!" para mostrar a las administraciones las ventajas de usar Software Libre. Sin embargo, todavía falta un lugar común para intercambiar este software dentro de las instituciones públicas en Alemania", explica Alexander Sander, Gestor de Política para la UE de la FSFE.

En consecuencia, "Un lugar para el código público" persigue el siguiente enfoque: El Software Libre para la administración se debe poder encontrar de forma estructurada y debe de ser posible desarrollarlo y documentarlo de forma colaborativa y al mismo tiempo cumplir con la ley. Además, es importante que el entorno promueva el intercambio entre las partes interesadas, como programadores, usuarios y administradores, y apoyar la creación de redes para que puedan crearse comunidades en torno a proyectos individuales.

Para cumplir estos requisitos, se necesita un catálogo de soluciones de Software Libre, una administración de usuarios federada, un código y una plataforma de información. Ya existen soluciones de Software Libre para este propósito, que pueden ser desplegadas en un primer paso. El siguiente paso es ahora encontrar una organización de apoyo y asegurar una financiación estable para satisfacer las necesidades de los interesados que utilicen esta plataforma en el futuro.

La iniciativa "Un lugar para el código público" invita a las administraciones, políticos y comunidad del Software Libre a participar en el debate y a seguir desarrollando el proyecto, que puede acelerar considerablemente la digitalización de la administración, en consonancia con la idea del Software Libre.

La documentación básica de este proyecto se puede encontrar aquí (.pdf).

La iniciativa "¿Dinero Público? ¡Código Público!" tiene como objetivo establecer el Software Libre como un estándar para el software financiado públicamente. Las administraciones públicas que siguen este principio pueden beneficiarse de numerosas ventajas: cooperación con otros organismos gubernamentales, independencia de los proveedores individuales, ahorro potencial de impuestos, promoción de la innovación y una base más sólida para la seguridad de la tecnología de la información.

La iniciativa "¿Dinero público? ¡Código Publico!" de la Fundación del Software Libre de Europa está apoyada por más de 180 organizaciones y administraciones, incluyendo la ciudad de Barcelona. Para saber más, por favor, visita: publiccode.eu/
