Ελευθερώστε το Android σας

Συμβάλλετε και βοηθήστε και άλλους!

Βοηθήστε μας να ενθαρρύνουμε τους ανθρώπους να απελευθερώσουν τις συσκευές τους και να πείσουμε τους προγραμματιστές εφαρμογών να εκδώσουν τις εφαρμογές τους ως Ελεύθερο Λογισμικό. Οι διάφορες πρωτοβουλίες Ελεύθερου Λογισμικού χρειάζονται επίσης βοήθεια και χρειάζεται να διασφαλίσουμε ότι παραμένουν ελεύθερες όπως στην ελευθερία. Αν θέλετε να διαδώσετε το μήνυμα αυτής της δράσης, υπάρχει διαφημιστικό υλικό διαθέσιμο για λήψη, μετάφραση και παραγγελία.

Λειτουργικά συστήματα

Η πρωτοβουλία Replicant βρίσκεται σε επείγουσα ανάγκη συνδρομής. Στην αρχική σελίδα της παρουσιάζει πολλούς τρόπους για να βοηθήσετε.

Αν και άλλες προσαρμοσμένες εκδόσεις Android είναι όλες βασισμένες στο Ελεύθερο Λογισμικό, μερικές φορές μη-ελεύθερα τμήματα προστίθενται σε αυτές τις μνήμες ROM. Αν γνωρίζετε για προσθήκες μη-ελεύθερου λογισμικού στο CyanogenMod ή σε άλλες εκδόσεις του Android, γνωστοποιήστε το σε εμάς! Συλλέγουμε τις τρέχουσες πληροφορίες στο wiki.

Εφαρμογές και F-Droid

Η πρωτοβουλία F-Droid περιέχει πολλούς τρόπους συνεισφοράς. Ρίξτε μια ματιά! Ίσως να υπάρχει κάτι που μπορείτε να κάνετε εύκολα.

Εκτός από αυτό, μπορείτε να βοηθήσετε να γίνει προεγκατάσταση στο CyanogenMod του F-Droid ώστε περισσότεροι να επωφεληθούν από μια ελεύθερη αγορά. Άλλες προτάσεις για να διαδώσετε το μήνυμα για το F-Droid μπορούν να βρεθούν στο φόρουμ του F-Droid.

Προσθέστε εφαρμογές μόνοι σας

Ενθαρρύνεστε να προσθέσετε εφαρμογές στο F-Droid. Παρέχεται ένα εγχειρίδιο για να σας βοηθήσει. Υπάρχει μια μεγάλη λίστα εφαρμογών σε αναμονή να προστεθούν. Η προσθήκη λειτουργεί παρόμοια με τις FreeBSD ports και τα gentoo ebuilds με γέμισμα ενός απλού αρχείου συνταγής που ελέγχει πώς ένα πακέτο κατασκευάζεται από πηγαίο κώδικα. Αν σας αρέσει να προσθέτετε εφαρμογές, πρέπει να διαβάσετε σχετικά με το πώς το F-Droid λειτουργεί με το git και πώς να γράφετε αυτά τα λεγόμενα αρχεία μεταδεδομένων.

Απελευθερώνοντας μη-ελεύθερες εφαρμογές

Δυστυχώς, υπάρχουν ακόμη μερικές εφαρμογές για τις οποίες δεν υπάρχει ελεύθερη εναλλακτική. Η χρήση ιδιοκτησιακής εφαρμογής είναι κακή ιδέα· αφαιρεί την ελευθερία σας. Επιπλέον, αν συνηθίσετε το μη-ελεύθερο πρόγραμμα, δεν θα νιώσετε την ανάγκη να γίνει ελεύθερο. Ακόμη και αν δεν γνωρίζετε προγραμματισμό, μπορείτε να έρθετε σε επαφή με τους συγγραφείς κρίσιμων εφαρμογών και να τους ρωτήσετε ευγενικά τους λόγους για τους οποίους δεν εκδίδουν την εφαρμογή τους ως Ελεύθερο Λογισμικό. Πολλές εφαρμογές είναι διαθέσιμες χωρίς πληρωμή, έτσι το χρήμα δεν είναι πάντα η κύρια αιτία. Μερικές φορές ένα μήνυμα μπορεί να κάνει τη διαφορά και υπάρχουν ήδη επιτυχημένες προσπάθειες: το Chess Walk για παράδειγμα έχει τώρα απελευθερωθεί.

Εφαρμογές που ακόμη χρειάζεται να απελευθερωθούν

Συλλέγουμε εφαρμογές στο wiki μας που δεν έχουν ελεύθερο ισοδύναμο, αλλά που η παρουσία τους είναι σημαντική σε ένα ελεύθερο σύστημα. Προσθέστε εκεί άλλες βασικές εφαρμογές. Οι άνθρωποι μπορούν να έρθουν σε επαφή με τους συγγραφείς αυτών των εφαρμογών και να τους ρωτήσουν γιατί αυτές δεν είναι Ελεύθερο Λογισμικό.

Απαντήσεις σε ενστάσεις που έχουμε ακούσει από προγραμματιστές εφαρμογών

Μόνο όταν κατανοήσουμε το κίνητρο των προγραμματιστών που διατηρούν τις εφαρμογές τους ιδιοκτησιακές, γνωρίζουμε τι χρειάζεται να αλλάξει για να σκεφτούν διαφορετικά και να τους πείσουμε να απελευθερώσουν τη δική τους εφαρμογή. Συνεισφέρετε τις εμπειρίες σας εδώ με προγραμματιστές μη-ελεύθερων εφαρμογών και μοιραστείτε χρήσιμες απαντήσεις στις ενστάσεις τους.

  • Ένσταση: Αν δημοσιεύσω την εφαρμογή μου ως Ελεύθερο Λογισμικό, δεν θα μπορώ να βγάλω χρήματα από αυτήν.
    • Μπορείτε να προσφέρετε την εφαρμογή σας σε άλλες αγορές και να ζητήσετε ανταμοιβή εκεί.
    • Προσφέρετε επιλογές δωρεάς. Οι ελεύθερες αγορές όπως η F-Droid θα τις συμπεριλάβουν και οι χρήστες τους θα ανταποκριθούν. Ανάλογα με την περίπτωσή σας, ίσως να βγάλετε περισσότερα με δωρεές παρά με άλλες πωλήσεις.
  • Ένσταση: Αλλά όταν η εφαρμογή μου είναι Ελεύθερο Λογισμικό άλλοι θα τη διανέμουν χωρίς εγώ να έχω κέρδος.
    • Αυτό είναι αλήθεια, αλλά σκεφτείτε ότι όσοι δεν θέλουν να σας πληρώσουν θα βρίσκουν πάντα τρόπους να το κάνουν.
    • Αν το ζητήσετε με ένα φιλικό διακριτικό τρόπο και με σωστή αιτιολόγηση, θα σας δώσουν χρήματα.
    • Αν δημοσιεύσετε την εφαρμογή σας με μια copyleft άδεια χρήσης όπως την GNU GPL, τότε έχετε εγγύηση ότι άλλοι δεν μπορούν να τη διανείμουν χωρίς να διανείμουν και τον πηγαίο κώδικα. Διαφορετικά πρόκειται για παραβίαση πνευματικών δικαιωμάτων και μπορείτε να επικοινωνήσετε με τον χειριστή του καταστήματος εφαρμογών για να την αφαιρέσει.
    • Μπορείε να πείτε στον κόσμο ότι για να αναγνωρίσουν και να υποστηρίξουν τη δουλειά σας, πρέπει να αγφοράσουν την εφαρμογή σας και όχι τα άλλα αντίγραφα που ίσως να διανέμονται από άλλους.
  • Ένσταση: Είμαι σε δύσκολη θέση εξαιτίας του άσχημου κώδικα που έγραψα και δεν θέλω κανείς να τον δει (από Flattr4Android).
    • Ο κώδικα σας είναι πολύ χρήσιμος σε πολλούς και λειτουργεί. Βέβαια, ένας καλός κώδικας είναι προτιμότερος, αλλά η ποιότητα του κώδικα είναι δευτερεύουσας σημασίας σε σχέση με την ελευθερία. Your code is very useful to many people and it works. Of course
    • Όλοι γράφουν κακό κώδικα κάποια στιγμή. Η κοινή χρήση του θα επιτρέψει σε άλλους να σας βοηθήσουν να τον βελτιώσετε.

Επικοινωνία

Αν θέλετε να παραμείνετε ενήμερος ή να συνεργαστείτε με άλλους σε ένα ελεύθερο σύστημα Android, μπορείτε να εγγραφείτε στη λίστα αλληλογραφίας. Μπορείτε επίσης να υποβάλετε ερωτήσεις ή να δώσετε τη γνώμη σας με γραπτό μήνυμα στο android@lists.fsfe.org. Για τεχνικές ερωτήσεις σχετικά με εφαρμογές, ο δίαυλος IRC για το F-Droid είναι μια καλή επιλογή για ερωτήσεις. Για συζήτηση σχετικά με τα κατώτερα επίπεδα του Android, το Replicant και ο δίαυλος IRC για το CyanogenMod είναι τα πιο κατάλληλα. Όλοι στην κοινότητα αυτή είναι πολύ φιλικοί και τους αρέσει να ανταποκρίνονται στην προσφορά βοήθειας, οπότε μην είστε ντροπαλοί και ασχοληθείτε!
