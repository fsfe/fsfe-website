Vantagens

As escolas devem ensinar os filhos a serem bons membros da comunidade, e as escolas não devem fornecer formação em produtos de empresas que não respeitam a liberdade.

  • Partilhar: As escolas deveriam ensinar as crianças a ajudar uns aos outros e partilhar uns com os outros. Com o software privativo, os professores são requeridos para evitar a partilha e dizer às crianças que o compartilhamento de software é errado. Utilizar o Software Livre permite que as escolas ofereçam um melhor exemplo e ensinar as crianças a compartilhar e colaborar e, assim, juntar toda uma comunidade que compartilha conhecimento.
  • Igualdade na casa: Com o Software Livre, os professores podem dar uma cópia para cada aluno. Assim, os pais não ficam em posição de tomar uma decisão financeira e as crianças de famílias com menos recursos financeiros podem aprender com as mesmas ferramentas que qualquer outra criança.
  • Aprender a programar: Algumas crianças estarão interessadas em como o software funciona, e alguns estarão interessados em escrever software. Quando uma escola usa Software Livre, eles estão em posição para ajudar as crianças a aprender sobre os computadores em profundidade.
    A possibilidade/liberdade de mexer motiva as crianças a aprender mais.
  • Aprender a usar o software: Hoje em dia não é o suficiente saber como usar um programa de escritório. Os trabalhadores jovens precisam de capacidade para usar qualquer software. É importante compreender os conceitos subjacentes a toda uma categoria ou tipo de software (como uma planilha ou um processador de texto) e não apenas como usar um aplicativo específico. A variedade de produtos de Software Livre ensinam exatamente isso. Para uma melhor compreensão, a comparação dos diferentes conceitos e pontos de vista (por exemplo, LyX / LaTeX vs Open Office) tem uma valia inestimável. O Software Livre permite que os alunos aprendam como funciona o software e, assim, como eles podem fazer o melhor uso dele. Uma habilidade chave, exigida pelos empregadores.
  • Dependência: Se as crianças aprendem a fazer as coisas de uma certa maneira na escola, a maneira mais fácil para eles fazer aquela coisa na vida adulta é continuar a fazê-lo da mesma maneira. Se as escolas ensinam os filhos a confiar no software privativo, estão dando à criança uma dependência de algo que eles têm que pagar e que, em geral, desestimula a partilha e a boa vontade na sociedade. No entanto, se uma escola ensina as crianças a confiar no Software Livre, o software não pode ser tirado da criança (mesmo na sua vida de adulto) e a criança pode continuar usando este software, além de ajudar os outros, partilhando-o.
    Usar e ensinar com o Software Livre faz também a própria escola independente de quaisquer interesses comerciais
  • Nenhum problema com as licenças: Com o Software Livre, não existem preocupações com as datas de validade ou os custos das licenças. O Software Livre permanece livre em todos os sentidos. Não há risco de induzir a criança a usar uma cópia ilegal (ver também: "A igualdade na casa").
  • Fácil de administrar: Há várias soluções prontas para o uso, para muitos casos em cenários educacionais. A manutenção dos sistemas de Software Livre é altamente automatizada e, portanto, economiza tempo. A segurança e outras atualizações podem ser implementadas em um minuto.
    O Software Livre é estável, seguro e confiável.
  • Sem taxas de licença O Software Livre pode economizar o dinheiro da escola. Como não há taxas de licença, o dinheiro pode ser usado para treinar os professores ou para o apoio técnico para descarregar os professores de trabalho.
  • Melhor uso de hardware antigo O Software Livre pode ser usado com baixos requisitos de hardware em quase todo o hardware. Assim, poupa dinheiro novamente.
  • Modificável, aprendizagem na língua materna: Os aplicativos de Software Livre estão sendo traduzidos a quase todas as línguas (até mesmo para aquelas onde a tradução não poderia ser oferecida por um desenvolvedor de software privativo). Assim, cada criança pode aprender e concentrar-se nas suas tarefas sem nenhuma barreira linguística. Pode até ser adaptada às características culturais locais
