A GNU/Linux Audio distribution
(IST-2001-34879)

One seminal role of the FSF Europe inside AGNULA is to take care of the legal issues, decide upon acceptable licensing policies and care about legal maintainability. Other tasks are making sure that Free Software community interests are heard, helping that information is provided in the best possible way and work on the dissemination of the final results.

FSF Europe Contributions

  • Deliverable 1.3.1: "Free Software Copyright and Licensing Recommendations"
    Available as HTML and PDF (57k)
  • Fiduciary Licence Agreement (FLA)
    The implementation of the policy described in deliverable 1.3.1
  • Python GUI for jMax
    A description of the problems with the Java based GUI of jMax and call for volunteers.

Status

The current biggest problem is still the Java dependency of the jMax GUI. Help creating an entirely Free Software GUI for jMax is very much appreciated. See this page for more information.
