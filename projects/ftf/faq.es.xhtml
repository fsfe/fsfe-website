¿Qué es la Freedom Task Force?

Es un protector para el Software Libre. La Freedom Task Force ofrece servicios de soporte sobre licencias a particulares, proyectos y negocios relacionados con Software Libre. Estamos aquí para ayudar a consolidar la construcción legal del ecosistema de Software Libre

¿Qué servicios ofrece la Freedom Task Force?

  • Actividades fiduciarias
  • Violación de licencias
  • Formación en materia de licencias

¿Cómo puedo contactar a la Freedom Task Force?

  • Puede enviar un mensaje a través de nuestro formulario en línea
  • Nos puede enviar un correo electrónico a ftf@fsfeurope.org

¿Qué conexión tiene con gpl-violations.org?

Trabajamos en cooperación con gpl-violations.org para proteger el Software Libre.

¿Qué conexión tiene con el FSF License Lab?

>Todas las fundaciones de Software Libre cooperan para asegurar la promoción y protección de Software Libre.

¿Qué conexión tiene con el Software Freedom Law Center?

Perseguimos el mismo objetivo de promover y proteger el Software Libre. Cooperaremos en la medida de lo posible para asegurar la promoción y la protección de Software Libre.
