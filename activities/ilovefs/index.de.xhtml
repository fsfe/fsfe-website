3 Ich Liebe Freie Software Tag

Ich Liebe Freie Software Tag

Wir unterschätzen oft die Kraft eines einfachen Dankes. All diejenigen welche sich an Freien Software beteiligen oder sich für diese einsetzen, leisten wertvolle Arbeit für unsere Gesellschaft und verdienen Aufmerksamkeit. Der "Ich Liebe Freie Software Tag" am 14. Februar ist die perfekte Gelegenheit für dich, deine Dankbarkeit auszudrücken.

Stell dir eine Freie Software Community vor, welche neue und alte Mitwirkende willkommen heißt und wertschätzend miteinander umgeht. Diese Gemeinschaft ist motivierend und freundlich im Umgang miteinander. Stell dir vor jemand Neues oder Altes möchte Teil des Freien Software Projekts sein und schickt eine Mail an die Mailingliste mit einer neuen Idee. Die Antworten auf diese Mail wären dankbar, freundlich und voller Wertschätzung für die Arbeit des Mitwirkenden. Wenn ein Pull Request bearbeitet wird, heben die Reviewer ganz automatisch hervor, wie dankbar sie für die Arbeit hinter dem Pull Request sind. Stell dir solche eine Freie Software Community vor, welche dankbar ist für deren Mitwirkende, Entwickler*innen und Maintainers. So eine Community lässt sich leicht realisieren mit einem Dankeschön. Danke das du Teil von Freier Software bist!

Je größer ein Freies Software Projekt wird, je mehr Mitwirkende es hat, desto schwieriger ist dieses zu pflegen. Am "I Liebe Freie Software Tag" schaffen wir einen Tag, welcher all diese Arbeit sieht und wertschätzt. Indem wir zu einander Danke sagen! Zu all den Leuten hinter Freien Software Projekten, zu all den Leuten die sich für Freie Software stark machen: Danke ♥

Indem wir Danke zueinander sagen schaffen wir eine offene Gemeinschaft, in welcher sich nicht nur erfahrene Enthusiasten sicher fühlen, sondern auch neue Leute ermutigt werden, sich zu beteiligen. Ein einfaches Dankeschön kann andere dazu motivieren auch weiterhin Teil von Freier Software zu sein.

Mit einem einfachen Dankeschön zeigen wir am 14. Februar, dass wir den Aufwand sehen den Mitwirkende für Freie Software leisten.

Denn Freie Software ist nicht irgendeine Software: Freie Software steht für eine Community, Freie Software ist Software, welche uns ermöglicht die Software zu verwenden, zu verstehen, zu teilen und zu verbessern. Lasst uns gemeinsam eine Freie Software Community schaffen welche einander wertschätzt und willkommen heißt, indem wir die Software feiern, welche uns ermächtigt.

Viele Menschen sind aus unterschiedlichen Gründen von der Freien Software begeistert.

Am alljährlichen 'Ich Liebe Freie Software Tag' bringen wir unsere Dankbarkeit gegenüber allen Mitwirkenden an Freier Software zum Ausdruck. Du kannst Teil von diesem besonderen Tag sein und damit auch Teil einer wundervollen Gemeinschaft! #ilovefs
