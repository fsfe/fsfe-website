Donazioni Hardware

La Free Software Foundation Europe vuole ringraziare tutti coloro che hanno donato delle apparecchiature per il loro supporto, che ci permette di continuare a lavorare per il Software Libero.

Se siete interessati a donare attrezzature, potete scrivere a office@fsfeurope.org specificando il tipo di apparecchiature che avete intenzione di donare. Potete aiutarci anche con donazioni in denaro o collaborando con noi.

Se state cercando dei loghi promozionali, li potete trovare qui.

  • 17/01/2005: Hewlett-Packard Germany ha donato due server Compaq N2400 (Dual CPU, Pentium III 1GHz con 1GB RAM) che stiamo usando come server web e mail.
