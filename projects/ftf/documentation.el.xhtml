Θέλουμε να βοηθήσουμε τους χρήστες να κατανοήσουν την αδειοδότηση του Ελεύθερου Λογισμικού, και μια σημαντική πτυχή αυτής της κατανόησης είναι η παροχή τεκμηρίωσης που μπορεί να είναι χρήσιμη. Αυτή η ιστοσελίδα είναι μια διαρκής προσπάθεια για την παροχή σαφούς πληροφόρησης σχετικά με διάφορες πτυχές της αδειοδότησης Ελεύθερου Λογισμικού. Παρακαλούμε να επιστρέφετε συχνά εδώ για ενημερώσεις και προσθήκες.
