Lista med 2005 års sponsorer

Free Software Foundation Europe vill tacka alla medhjälpare, bidragare och andra personer som har hjälpt oss att arbeta med och för fri programvara. Information om hur du kan stödja oss i vårt arbete för fri programvara kan du hitta här. Information för eventuella sponsorer finns också.

Free Software Foundation Europe vill med namn tacka följande sponsorer. Ert stöd gjorde det möjligt för oss att fortsätta jobba för fri programvara.

Vi vill också tacka alla sponsorer som önskat vara anonyma. Om du vill donera så hittar du all nödvändig information här. Söker du knappar för att visa att du stöder oss så finns de här.
