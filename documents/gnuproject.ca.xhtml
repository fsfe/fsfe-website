Què és el projecte GNU?

El projecte GNU es va iniciar al setembre de 1983 per l'americà dels EUA Richard M. Stallman per tal de crear un sistema operatiu complet d'estil Unix basat en Programari Lliure. Les feines van començar al gener següent.

Stallman va iniciar la "Free Software Foundation" en 1985 per a tractar amb els aspectes jurídics i organitzatius del projecte GNU i també per a estendre l'ús i el coneixement sobre el Programari Lliure. Mitjançant la Free Software Foundation s'han formulat les llicències GNU General Public License i la GNU Lesser General Public License (originalment anomenada GNU Library Public License) i amb els anys s'han establert com les llicències per a Programari Lliure més usades.

El projecte GNU el formen subprojectes més petits mantinguts per voluntaris o corporacions, normalment amb l'objectiu de crear o mantenir un component funcional. Aquests mateixos subprojectes es diuen també "GNU Projects" o "official GNU Projects."

El nom del projecte GNU deriva de l'acrònim recursiu "GNU's Not Unix" ("GNU No és Unix"). Com que Unix era originalment no un cert sistema, sinó més aviat un producte, això pretenia simbolitzar que el projecte GNU té l'objectiu de crear un sistema compatible amb, però no idèntic, a Unix.

El sistema GNU és (com altres sistemes operatius Unix) modular i avui, especialment el sistema GNU amb el nucli Linux - l'anomenat sistema GNU/Linux -, s'usa àmpliament i proporciona la base de cada anomenada "Distribució Linux".
