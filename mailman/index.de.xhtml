Über die Mailinglisten

Die Mailinglisten stehen jedem offen, der an der Free Software Foundation Europe interessiert ist.

Sowohl die Eintragung in die Mailinglisten als auch die komplette Verwaltung der Eintragungen kann entweder per E-Mail oder online über die Webschnittstelle erfolgen. Außerdem gibt es online vollständige Archive der Diskussionen.

Discussion

Discussion ist die Mailingliste für allgemeine, die FSF Europe betreffende, Diskussionen. Hier werden Einrichtung, Aufgabenbereiche und Organisation besprochen.

In Discussion können Sie sich entweder eintragen, indem Sie die Webschnittstelle nutzen oder eine E-Mail an Discussion-request@fsfeurope.org mit subscribe als Betreff schreiben.

Die Archive können hier gefunden werden.

Announce

Announce ist die Ankündigungsliste der FSF Europe. Wenn Sie auf dem aktuellen Stand über wichtige Schritte informiert bleiben wollen, ohne die Gespräche der normalen Diskussionsliste zu erhalten, ist dies die geeignete Liste für Sie.

Sie können sich zu Announce anmelden, indem Sie entweder die Webschnittstelle nutzen oder eine E-Mail an Announce-request@fsfeurope.org mit subscribe als Betreff schreiben.

Die Archive können hier gefunden werden.

Web

Dies ist die Liste für Diskussion über und Organisation der FSF Europe Webseiten. Auch E-Mails an webmasters@fsfeurope.org and webmaster@fsfeurope.org werden hierhin weitergeleitet.

Sie können sich zu Web anmelden, indem Sie entweder die Webschnittstelle nutzen oder eine E-Mail an web-request@fsfeurope.org mit subscribe als Betreff schreiben.

Die Archive können hier gefunden werden.

Web-cvs-notify

Dies ist die CVS Log Mailingliste. Sie ist nur für Webmaster interessant.

Sie können sich zu Web-cvs-notify anmelden, indem Sie entweder die Webschnittstelle nutzen oder eine E-Mail an web-cvs-notify-request@fsfeurope.org mit subscribe als Betreff schreiben.

Die Archive können hier gefunden werden.

press-release

An diese Liste kann nur von der FSFE geschrieben werden und sie enthält nur englische Pressemitteilungen.

Sie können press-release abonnieren, indem Sie entweder die Webschnittstelle nutzen oder eine E-Mail an press-release-request@fsfeurope.org mit subscribe als Betreff schreiben.

Die Archive können hier gefunden werden.

press-release-fr

An diese Liste kann nur von der FSFE geschrieben werden und sie enthält nur französische Pressemitteilungen.

Sie können press-release-fr abonnieren, indem Sie entweder die Webschnittstelle nutzen oder eine E-Mail an press-release-fr-request@fsfeurope.org mit subscribe als Betreff schreiben.

Die Archive können hier gefunden werden.

press-release-de

An diese Liste können nur Mitglieder der FSFE E-Mails schreiben. Sie dient ausschließlich der Verbreitung von Pressemitteilungen.

Sie können press-release-de abonnieren, indem Sie die Webschnittstelle nutzen, oder indem Sie eine E-Mail an press-release-de-request@fsfeurope.org mit dem Betreff subscribe schicken.

Die Archive dieser Mailingliste können hier gefunden werden.

E-Mail-Listen auf Savannah

Offene Mailinglisten auf Savannah ermöglichen es jedem (Mitglieder der FSFE, Mitglieder der assoziierten Organisationen, Freiwillige), sich mit der FSF Europe an der Freien Software-Bewegung zu beteiligen.
