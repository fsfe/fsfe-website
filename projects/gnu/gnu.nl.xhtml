GNU's Not Unix!


Het GNU project, reeds in 1983 aangekondigd door Richard M. Stallman, startte in 1984 met het ontwikkelen van een Vrije Software implementatie van een unix-achtig besturingssysteem. Verschillende varianten van dit systeem die gebruik maken van een linux kernel, worden nu over heel de wereld veel gebruikt. Deze GNU/Linux systemen vormen de basis voor alle moderne "Linux" distributies.

Binnen het kader van het GNU project werd in 1985 de Free Software Foundation opgericht. Haar oorspronkelijke taak was de juridische ondersteuning van het GNU project.

De FSF Europa ondersteunt het werk van de FSF Noord-Amerika voor het GNU project, zowel op juridisch als op technisch gebied.

De webpagina's van het GNU project vind je op 

   http://www.gnu.org
De webpagina's van de Free Software Foundation in Boston vind je op
   http://www.fsf.org
