The FSF* network

Trabajamos con otros grupos internacionales para promover el Software Libre en todos los continentes. Las siguientes organizaciones son organizaciones hermanas de la FSFE, y forman parte de la red de Software Libre. Cada organización tiene su propia cultura y es legal, financiera y operativamente independiente.

  • 1985: La Free Software Foundation (FSF) se crea poco después del proyecto GNU.
  • 2001: Se funda la Free Software Foundation Europa (FSFE).
  • 2003: Se funda la Free Software Foundation India(FSFI).
  • 2005: Se funda la Free Software Foundation Latin America (FSFLA).
