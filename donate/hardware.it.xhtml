Donazioni di hardware ed infrastrutture

La Free Software Foundation Europe ringrazia tutti i donatori di apparecchiature e servizi per il loro aiuto concreto: i sistemi che utilizziamo sono essenziali per continuare a lavorare a favore del Software Libero. In particolare vorremmo ringraziare:

  • 08-10-2020: FOSSHOST
    Per la donazione di una prestante macchina virtuale per le API di REUSE.
  • 07-05-2020: Passbolt
    Per aver fornito una licenza Pro senza limiti per l'istanza sui nostri server.
  • 08-12-2016: PlusServer
    Per la donazione di una cospicua quantità di RAM destinata ad uno dei nostri server fisici.
  • 15-10-2014: abaton EDV-Dienstleistungs GmbH
    Per la donazione di due server fisici.
  • 11-08-2014: OpenIT
    Per la donazione di tre server fisici destinati a rimpiazzarne uno donato dalla stessa società in precedenza e per l'hosting degli stessi.
  • 11-04-2014: kapper.net
    Per aver fornito l'housing ad uno dei nostri server fisici.
  • 11-10-2013: abaton EDV-Dienstleistungs GmbH
    Per la donazione di un server fisico.
  • 23-02-2011: Kolab Systems AG
    Per l'assistenza fornita durante l'installazione e la configurazione iniziale del sistema Kolab ad uso interno della FSFE.
  • 09-11-2010: CapeIT
    Per la donazione di server fisici, NAS e di una nastroteca per backup.
  • 05-12-2009: OpenIT
    Per la messa a disposizione di tre server fisici presso il proprio datacenter, dai quali sono ora forniti molti dei servizi della FSFE ed inoltre per la fornitura di una connessione veloce per gli stessi.
  • 25-09-2009: antilo AG
    Per la donazione di un server virtuale destinato ad essere usato come server VoIP interno.
  • 20-09-2009: AMOOMA GmbH
    Per l'assistenza fornita durante l'installazione e la configurazione iniziale del nostro server VoIP.
  • 01-09-2009: beroNet
    Per la donazione di cinque telefoni VoIP destinati alla nostra sede berlinese.
  • 21-03-2007: Sun Microsystems
    Per la donazione di un server Sun Fire T1000 (8 CPU 4-core, 16GB RAM, 2 hard disk da 73GB). FSFE ha utilizzato questo server per i servizi email e della Fellowship.
  • 20-05-2006: Dipartimento di Informatica, Università di Göteborg, Svezia
    Per la donazione di un computer Dual Pentium III 1GHz con 2GB RAM e di un disk cabinet IBM EXT300 con 14 dischi da 18 GB.
  • 27-12-2005: SAMSUNG Electronics Germania
    Per la donazione di sei portatili SAMSUNG X20 XVM 1730 V che vengono utilizzati per il lavoro della FSFE ad Amburgo, Göteborg, Milano, Bruxelles, Francoforte e Vienna; inoltre per la donazione di una macchina fotografica digitale SAMSUNG Digimax V700.
  • 17-01-2005: Hewlett-Packard Germania
    Per la donazione di due server Compaq N2400 (Dual CPU, Pentium III 1GHz con 1GB RAM). Le macchine hanno sostituito i nostri due vecchi web e mail server, e ora forniscono servizi essenziali per la Free Software Foundation Europe.
  • 24-01-2003: Dipartimento di Informatica, Università di Göteborg, Svezia
    Per la donazione di un disk cabinet Andataco con 3 dischi da 18 GB e 5 da 9 GB, un computer Pentium III a 600MHz, un Pentium III a 450 MHz ed un Dual Pentium II a 450 MHz.

    • (lista ordinata per data, le donazioni più recenti sono all'inizio)
