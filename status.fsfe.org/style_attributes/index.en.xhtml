1 FSFE Syle Attribute Status

Style Attribute Status

Here we have the output of a simple script to generate a log of what files have html style elements in them.

The aim of this is to make it easier to have some idea of what files we should be working on.

Now, a brief explanation of each column

File
The filepath of the file, linked to the file in the git repo
Element
The XML element that has a style attribute
Style Value
The style value of the element
