GPLv3: Entwurf der 3. Version der GNU General Public License

Wie befinden uns in einem einjährigen öffentlichen Beratungsprozess zum Entwurf der dritten Version der GNU General Public License. Die Free Software Foundation Europe unterstützt diesen Prozess indem sie das Bewusstsein für die vorgeschlagenen Änderungen schärft und versucht die öffentliche Beteiligung an diesem Prozess zu maximieren. Einführung: Was bisher geschah

Am 16. Januar 2006 veröffentlichte die Free Software Foundation (FSF), unsere Schwesterorganisation in den Vereinigten Staaten, den ersten Entwurf der GPL Version 3 und startete damit eine Beratungsphase, welcher ungefähr ein Jahr andauern wird.

Am Ende dieser Beratungsphase wird die FSF die aktualisierte GNU General Public License veröffentlichen. Diese Lizenz, gewöhnlich "die GPL" genannt, ist die bei weitem meistgenutzte Lizenz für Freie Software.

Die Aufgabe der GPL ist es jedem, der eine Kopie GPL-lizenzierter Software bekommt, die Freiheit zu geben sie zu verwenden, zu verändern und weiter zu geben. Heute verwenden einige hundert Millionen Anwender Software, welche unter der GPL lizenziert ist.

Version 2 der GPL wurde 1991 veröffentlicht. Version 3 der GPL wird die gleiche Aufgabe erfüllen wie Version 2, aber an die juristischen und technischen Änderungen im Umfeld von Software angepasst sein und neuen Techniken entgegen wirken, welche in den letzten fünfzehn Jahren entwickelt wurden um den Anwendern von Software die oben genannten Freiheiten zu entziehen.

Wie Sie daran teilhaben können

Die öffentliche Beratung wird von unserer Schwesterorganisation auf der offiziellen GPLv3 Webseite organisiert. Dort können Sie:

  • Bereits vorhandene Kommentare einsehen
  • Sich anmelden
  • Sich einloggen
  • Kommentare abgeben (nachdem Sie sich eingeloggt haben)

Mit ihrer Anmeldung werden Sie auch in eine Mailingliste eingetragen, über die Sie unregelmäßig Informationen über zentrale Veranstaltungen zur GPLv3 erhalten.

Zeitplan

Im Folgenden finden Sie einen Zeitplan für den gesamten Prozess mit Informationen zu europäischen GPLv3 Veranstaltungen.

16. Januar 2006
Der erste öffentliche Entwurf wurde zusammen mit einem Grundlagendokument veröffentlicht, das die Änderungen erklärt. Dies markiert den Beginn der öffentlichen Beratungsphase. Dies geschah während der ersten internationalen GPLv3 Konferenz.
25. Februar 2006
Belgien: Richard Stallman hält eine Präsentation zur GPLv3 auf der FOSDEM ( Mitschrift)
18. März 2006
Italien: Richard Stallman gab in Turin (Italien) auf der Italienischen GPLv3 Konferenz eine Präsentation zur GPLv3. (Mitschrift der Präsentation)
21.-22. April 2006
Zweite internationale GPLv3 Konferenz, FISL, Porto Alegre (Brasilien).
29. Mai 2006
Georg Greve hielt einen Vortrag in Manchester: "Update of the constitution: introduction to GPLv3". Weitere Informationen finden Sie auf der externen Webseite zur Veranstalung (Englisch).
22.-23. Juni 2006
Dritte internationale GPLv3 Konferenz in Barcelona (Spanien).
28. Juni 2006, 16:00Uhr
Ciaran O'Riordan hält auf der GUADEC einen Vortrag: "GPLv3 and free software development"
Juli 2006 (voraussichtlich)
Nach 7 Monaten voller öffentlicher Kommentare und Diskussionen wird ein zweiten Entwurf veröffentlicht.
22.August 2006
Jonas Öberg wird auf dem Dataföreningen Region West, Göteborg (Schweden) einen Vortrag zur GPLv3 halten.
23.-24. August 2006
Vierte internationale GPLv3 Konferenz in Bangalore (Indien).
24. August 2006
Jonas Öberg und Mathias Klang halten einen Vortrag über die GPLv3 auf dem Dataföreningen Region East, Linköping (Schweden)
Oktober 2006 (voraussichtlich)
Fünfte internationale GPLv3 Konferenz, im fernen Osten.
15. Januar 2007
Wahrscheinliches Datum für die Veröffentlichung der offiziellen GPLv3.
März 2007
Spätester Zeitpunkt für die Veröffentlichung der offiziellen GPLv3.
