FTF: Πρόγραμμα Εμπιστευτικότητας

Η FTF δρα ως νομικός θεματοφύλακας για προγράμματα Ελεύθερου Λογισμικού. Το Πρόγραμμα Εμπιστευτικότητας του FSFE επιτρέπει σε έργα να χρησιμοποιήσηουν το Συμφωνητικό Εμπιστευτικότητας Άδειας Χρήσης (FLA) για να μεταβιβάσουν πνευματικά δικαιώματα στο FSFE, επιτρέποντας την ενοποίηση και την αποτελεσματική διαχείρισή τους. Αυτό επιτρέπει στους προγραμματιστές να εστιάζουν στην ανάπτυξη καταπληκτικών εφαρμογών, χωρίς να ανησυχούν για τις νομικές πτυχές.

Η προσέγγισή μας μπορεί να συνοψιστεί σε μια πρόταση. Η FTF, χωρίς να γίνεται κανονιστική, εξασφαλίζει ότι τα προγράμματα βρίσκονται σε νομικά ισχυρή θέση· δεν υπαγορεύουμε την πολιτική των προγραμμάτων ή τη διαχείρισή τους. Η δουλειά μας είναι να εξασφαλίσουμε ότι μπορείτε να κάνετε τη δική σας.

Οι λεπτομέρειες της διαδικασίας επιλογής εμπιστευτικότητας και οι πολιτικές μας είναι διαθέσιμα για επισκόπηση.

Επίσης μπορείτε να προσαρμόσετε το Συμφωνητικό Εμπιστευτικότητας Άδειας Χρήσης (FLA) για να ενοποιήσετε τα πνευματικά σας δικαιώματα υπό την δική σας διοικητική κάλυψη. Διαθέτουμε μια ιστοσελίδα που περιέχει προσαρμόσιμες εκδόσεις του FLA εδώ.
