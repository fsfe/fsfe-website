Ο Κατάλογος με τους Δωροθέτες για το 2006

Το Ευρωπαϊκό Ίδρυμα Ελεύθερου Λογισμικού θα ήθελε να ευχαριστήσει όλους τους εθελοντές, τους συντελεστές και άλλους που μας έχουν βοηθήσει να εργαστούμε για το Ελεύθερο Λογισμικό. Πληροφορίες για το πώς μπορείτε να μας υποστηρίξετε στην εργασία μας υπάρχουν εδώ. Πληροφορίες για ενδεχόμενες δωρεές υπάρχουν επίσης διαθέσιμες.

Το Ευρωπαϊκό Ίδρυμα Ελεύθερου Λογισμικού θα ήθελε ονομαστικά να ευχαριστήσει τους ακόλουθους δωροθέτες. Η υποστήριξή σας έκανε εφικτή για εμάς τη συνέχιση της εργασίας για το Ελεύθερο Λογισμικό.

Θα θέλαμε επίσης να ευχαριστήσουμε όλους τους δωροθέτες που επιθυμούν να παραμείνουν ανώνυμοι. Αν και εσείς θέλετε να δωρίσετε, μπορείτε να βρείτε όλες τις απαραίτητες πληροφορίες εδώ. Αν αναζητάτε τα κουμπιά χορηγιών, αυτά υπάρχουν εδώ.

Δωρεές εξοπλισμού αναφέρονται στον Ιστότοπο Δωρεών Εξοπλισμού. Εκεί, μπορείτε επίσης να βρείτε πληροφορίες για το πώς μπορείτε να μας υποστηρίξετε με εξοπλισμό.
