About the Mailing Lists

The mailing lists are open to everyone interested in the Free Software Foundation Europe.

Subscription to any of the two mailing lists as well as complete maintenance of subscriptions can either be done by e-mail or online with the web interface. There are also complete archives of the discussions available online.

Discussion

Discussion is the mailing list for general discussions concerning the FSF Europe. This is where the setup, functionality, organization are being discussed.

You can subscribe to Discussion by using the web interface or sending e-mail to Discussion-request@fsfeurope.org with subscribe as the subject.

The archives can be found here.

Announce

Announce is the announcement-only mailing list for the FSF Europe. If you want to be kept up to date on important steps without receiving the chatter of a normal discussion list, this list is for you.

You can subscribe to Announce by using the web interface or sending e-mail to Announce-request@fsfeurope.org with subscribe as the subject.

The archives can be found here.

Web

This is the list for discussions and the organisation of the FSF Europe web pages. Mail to webmasters@fsfeurope.org and webmaster@fsfeurope.org gets redirected here, as well.

You can subscribe to Web by using the web interface or sending e-mail to web-request@fsfeurope.org with subscribe as the subject.

The archives can be found here.

press-release

This list is post-only by the FSFE and contains only press releases in English. The subscriber list is non public.

You can subscribe to press-release by using the web interface or sending e-mail to press-release-request@fsfeurope.org with subscribe as the subject.

The archives can be found here.

press-release-fr

This list is post-only by the FSFE and contains only press releases in French. The subscriber list is non public.

You can subscribe to press-release-fr by using the web interface or sending e-mail to press-release-fr-request@fsfeurope.org with subscribe as the subject.

The archives can be found here.

press-release-de

An diese Liste können nur Mitglieder der FSFE E-Mails versenden. Sie dient ausschließlich der Verbreitung von Presse-Meldungen.

Sie können press-release-de abonnieren, indem Sie das Web-Interface verwenden, oder indem Sie eine E-Mail mit dem Betreff subscribe an press-release-de-request@fsfeurope.org senden.

Das Archiv diese Mailing-Liste finden Sie hier.
