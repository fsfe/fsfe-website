Liste des Supporters 2004

La Free Software Foundation Europe souhaite remercier tous les volontaires, contributeurs et autres personnes qui nous ont aidés à travailler pour le Logiciel Libre. Des informations sur comment vous pouvez soutenir notre travaille pour le Logiciel Libre sont disponibles à la page aide. Des informations pour les donateurs potentiels sont disponibles à la page dons.

La Free Software Foundation Europe voudrait remercier notamment remercier les donateurs ci-dessous. Votre soutien nous a permis de continuer à travailler pour le Logiciel Libre.

Nous souhaiterions également remercier tous les donateurs qui ont souhaité rester anonyme. Si vous aussi vous souhaitez faire un don, vous trouverez toutes les informations nécessaires à la page dons. Si vous recherchez les bannières de sponsors, vous les trouverez à la page bannières de sponsors.
